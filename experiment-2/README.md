#  Experiment 2: UI Design using Material UI (MUI) Components in React

This project is developed as part of **Experiment 2** for the course **Full Stack – II (23CSH-382)**.  
In this experiment, **Material UI (MUI) components** are integrated into the existing **React SPA created in Experiment 1** to design a modern and responsive user interface.

##  Experiment Details
- **Experiment No.:** 2  
- **Title:** Design UI using Component Library (Material UI)

##  Steps to Run the Project

1. Clone the repository  
`git clone <your-github-repository-url>`

2. Navigate to the project directory  
`cd my-spa`

3. Install dependencies  
`npm install`

4. Install Material UI  
`npm install @mui/material @emotion/react @emotion/styled`

5. Start the development server  
`npm run dev`

6. Open in browser  
`http://localhost:5173`
